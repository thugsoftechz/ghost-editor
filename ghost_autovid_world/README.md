
# GHOST_AUTOVID_WORLD

The World's Best Automated Video Editor.

## Usage

**GUI Mode:**
```bash
streamlit run ghost_autovid_world/app.py
```

**Headless Mode:**
```bash
python ghost_autovid_world/main.py /path/to/footage
```

## Features
- Offline, No API Keys
- Auto-Editing (Silence Removal, B-Roll, Mastering)
- Auto-Thumbnails & Metadata
- Hardware Accelerated
